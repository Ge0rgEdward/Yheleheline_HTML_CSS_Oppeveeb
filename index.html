MIS ON HTML?

HTML (HyperText Markup Language) on standardne märgistuskeel, mida kasutatakse veebilehtede loomisel. See võimaldab arendajatel struktureerida sisu ja lisada elemente, nagu pealkirjad, lõigud, pildid ja lingid.

HTML Põhitõed

HTML loob veebilehe struktuuri
Pealkirjad on <h1>–<h6>
Lõigud on <p> elemendis
Lingid luuakse <a> abil
Pildid lisatakse <img> elemendiga
Loendid: <ul> ja <ol>
Vormid kasutavad <form>, <input>, <button>

HTML Näited


<!DOCTYPE html>
<html lang="et">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>>Minu esimene veebileht</title>
</head>
<body>
    <h1>Tere tulemast!</h1>
    <p>See on minu esimene veebileht.</p>
</body>
</html>
